Any contact form plugin. Simple, free and flexible.

== Description ==

You can manage and edit all forms directly out of your WordPress Admin Center without any need of an external editor.

= Requirements =
1. WordPress version 3.7 and later
1. PHP Interpreter version 5.2.0 or later
1. PHP GD Library (if you want to use the captcha)

== Installation ==

1. Uncompress the download package
1. Upload folder including all files and sub directories to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress

You will find 'Any Contact' menu in your WordPress admin panel.

For basic usage, you can also have a look at the [plugin homepage](http://any-contact.com/).
